URL Tracker Manager
About
URL Tracker Manager is a local-first web app that lets users securely manage, categorize, and track website links, tags, UTM campaigns, and click stats entirely on their browser. It includes simple authentication, short links, export/import functionality, and privacy-focused storage without requiring any servers.

Features
User authentication with salted and hashed password storage (PBKDF2) done locally for enhanced privacy

Manage and categorize links with tags and categories

Track clicks on links with real-time statistics

Create and manage UTM campaigns for marketing tracking

Short links generation with optional slugs

Export and import link data for backup and transfer

Privacy-focused local storage; no server needed

Responsive and easy-to-use interface

Usage
Open the app in your browser (https://gamerayu25.github.io/URL_Tracker-Manager/).

Sign up by creating a secure account.

Sign in to access your link manager.

Add, categorize, and tag your links.

Track clicks and manage UTM campaigns.

Use export/import features to backup or transfer data.

Tech Stack
Built entirely in HTML/JavaScript for client-side execution

Local storage for data privacy

Note
This app is intended for local use only and not recommended for multi-user production environments.
